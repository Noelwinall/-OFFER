## 📐 设计方案

### 1. **所有页面底色统一**
- **颜色**: Ivy League Cream (#FAF8F5)
- **应用范围**: 首页、搜索页、收藏页、我的页面
- **优势**: 视觉一致性，无"跳跃感"

### 2. **首页差异化**
通过以下元素营造仪式感：

#### A. 深色渐变蒙层
```
从上到下逐渐透明：
深蓝 (92% opacity) → 中蓝 (70% opacity) → 浅蓝 (40% opacity) → 完全透明
```

#### B. 大气设计图元素
1. **📈 知识阶梯** (右下)
   - 4级台阶，象征教育进阶
   - 逐级上升，寓意成长路径

2. **🏛️ 学术殿堂柱子** (左侧)
   - 3根古典柱子 + 柱顶横梁
   - 象征知识殿堂、学术权威

3. **📚 书本堆叠** (右下角)
   - 3本书层叠摆放
   - 象征知识积累

4. **⭐ 成就之星** (右上)
   - 大星星 + 小星星点缀
   - 象征梦想与成就

5. **🔗 知识网络连接线** (中上部)
   - 虚线连接的节点
   - 象征知识网络

6. **精致线条装饰**
   - 右上角和左下角的细线装饰
   - 增添精致感

### 3. **其他页面保持简洁**
- **背景**: Dots 图案（极简）
- **目的**: 专注内容，不干扰阅读
- **体验**: 视觉轻盈，易于浏览学校信息

## 🔧 自定义调整示例

### 调整设计图透明度
```tsx
// 在 components/home-hero-background.tsx 中

// 阶梯
<G opacity="0.12">  // 增大 → 更明显，减小 → 更微妙

// 柱子
<G opacity="0.08">

// 书本
<G opacity="0.1">

// 星星
<G opacity="0.15">
```

### 调整蒙层深度
```tsx
colors={[
  "rgba(15, 22, 41, 0.92)",  // 顶部，增大 opacity → 更深
  "rgba(30, 58, 95, 0.85)",
  "rgba(44, 74, 110, 0.7)",
  "rgba(30, 58, 95, 0.4)",
  "rgba(250, 248, 245, 0)"   // 底部透明
]}
```
